## TypeScript
ECMAScript 6 with *optional typing*

## Static vs Optional Languages


#### Statically Typed Languages

Java
```java
String name = "Andrew";
```
C#
```cs
string name = "Andrew";
```
Objective-C
```
NSString name = "Andrew";
```

#### Optionally Typed Languages

TypeScript
```java
// optional!
var name: string = "Andrew";
// we can write instead
var name = "Andrew";
```
