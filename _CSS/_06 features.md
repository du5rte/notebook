
















## Cursors
specifies the mouse cursor displayed is over an element [MDN list of cursors](https://developer.mozilla.org/en-US/docs/Web/CSS/cursor)
http://www.sitepoint.com/css3-cursor-styles/
```css
div {
  cursor: pointer;
}

div {
  cursor: grab;
}
```
user-select: none;
pointer-events: none
